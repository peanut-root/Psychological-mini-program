# 如何预览小程序

## 🚀 快速开始

### 1. 下载微信开发者工具
- 访问：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
- 根据你的系统（macOS/Windows）下载对应版本

### 2. 打开项目
1. 启动微信开发者工具
2. 点击"导入项目"或"+"号
3. 选择项目目录：`/Users/peanut/Documents/PH IEP/11 上/心理小程序`
4. 填写项目信息：
   - **AppID**：选择"测试号"（无需注册）
   - **项目名称**：心屿心理科普小程序
5. 点击"导入"

### 3. 查看效果
- 工具会自动编译项目
- 左侧模拟器会显示小程序界面
- 可以点击不同页面查看效果
- 可以在调试器中查看代码和日志

## 📱 手机预览

1. 在开发者工具顶部点击"预览"按钮
2. 用微信扫描二维码
3. 在手机上查看小程序效果

## 🔧 调试技巧

- **控制台**：查看 `console.log` 输出
- **网络**：查看网络请求
- **存储**：查看本地存储数据
- **调试器**：可以设置断点调试

## ⚠️ 常见问题

1. **编译错误**：检查 `app.json` 中的页面路径是否正确
2. **页面空白**：检查页面 `.json` 文件配置
3. **样式不生效**：检查 `.wxss` 文件语法

## 📝 项目结构

```
├── app.js              # 小程序入口
├── app.json            # 全局配置（页面路径、TabBar等）
├── app.wxss            # 全局样式
├── pages/              # 所有页面
│   ├── index/          # 首页（主页面）
│   ├── my-page/        # 我的（TabBar页面）
│   ├── science/        # 科普（TabBar页面）
│   └── ...
```

## 🎯 主要页面

- **首页**：`pages/index/index` - 主界面，包含所有功能入口
- **TabBar页面**：可通过底部导航栏切换
  - 首页、科普、量表、互动、我的
