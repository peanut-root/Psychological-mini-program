# 如何预览小程序

## 快速开始

### 1. 下载微信开发者工具

访问：<https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html>  
按系统（macOS / Windows）下载并安装。

### 2. 打开项目

1. 启动微信开发者工具。  
2. 选择「导入项目」。  
3. 目录选择本仓库根目录（包含 `app.json` 的文件夹）。  
4. **AppID**：可选用「测试号」。  
5. **项目名称**：例如「心屿心理科普小程序」。  
6. 导入后等待自动编译。

### 3. 查看效果

- 左侧模拟器展示当前页面。  
- 在「编译」模式下修改 `pages/` 下文件并保存会触发热更新。  
- 可在调试器查看 `console`、Storage、网络等。

## 手机预览

1. 开发者工具顶部点击「预览」。  
2. 使用微信扫码。  
3. 在真机上操作以验证交互与布局。

## 调试提示

- **控制台**：查看 `console.log` 与报错栈。  
- **存储**：查看 `wx.setStorage` / `getStorage`（如文献点赞、评论缓存键 `literature:article:*`）。  
- **页面路径**：以 `app.json` 的 `pages` 为准；首页在列表中为 `pages/home/home`（非旧文档中的 `index`）。

## 常见问题

1. **编译报错**：检查 `app.json` 中页面路径是否存在对应四件套。  
2. **页面空白**：检查对应页 `.json` 是否合法、`.wxml` 是否有语法错误。  
3. **样式异常**：检查 `.wxss` 与全局 `app.wxss`、主题 `styles/theme.wxss` 的引入关系。

## 项目结构（与当前代码一致）

```
├── app.js / app.json / app.wxss   # 全局入口与样式
├── styles/theme.wxss              # 黄色主题等
├── images/                        # 图片资源
└── pages/
    ├── splash/ splash2/          # 引导（首次）
    ├── home/                     # 主页
    ├── literature/               # 文献列表（分类筛选）
    ├── literature-detail/        # 文献详情（?id=1..11，主路径）
    ├── literature-detail-1/ …   # 独立详情模板（备用）
    ├── science/ scale/ interaction/ my-page/ …
    └── …                         # 其余见 app.json
```

## 文献资料库说明

- **列表**：`pages/literature` — 含「全部」及抑郁症、焦虑症、双相、强迫、创伤应激、精神分裂、ADHD、物质滥用、恐惧症、**躯体化障碍**、**性偏好障碍**等分类。  
- **详情**：`pages/literature-detail` — 通过 URL 参数 `id` 加载第 1～11 篇白皮书体例正文（病名、俗称、症状、诊断、治疗、误解澄清、病例等）。  
- 点赞与评论数量为本地持久化后的展示，重进页面仍会读取最近一次数据。
